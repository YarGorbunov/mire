## Планируемый новый функционал
1 Фичи для взаимодействия игроков
1.1 Сообщения от игроков подписываются их именем ✔
1.2 look выводит инфу о игроках в комнате ✔
1.3 Возможность шепота сообщения определенному игроку (новая команда whisper, передающаяя сообщение только одному игроку в комнате)
1.4 Сообщения в чат игрокам если кто-то вошел/вышел в/из комнаты, в которой они находятся
1.5 Возможность передать предмет из инвентаря другому игроку
1.6 Возможность взять предмет в руки, чтобы другие игроки могли его видеть (новая команда)
1.7 Детектор находит предмет в инвентарях игроков
2 Фишки для подземелья
2.1 Новые комнаты
2.2 Рандомная генерация комнат (конечная или бесконечная, в целом пох, реализовать можно в любом виде)
